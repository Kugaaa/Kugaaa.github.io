发表于 2023-09-17 分类于 开发 > 网络

SSE

介绍

Server-Sent Events（SSE）是一种基于 HTTP 的服务器推送技术，它允许服务器向客户端实时发送数据

通过SSE，服务器可以主动推送数据给客户端，而无需客户端发起请求

SSE 建立在 HTTP 协议之上，使用了长连接（持久连接）来实现服务器和客户端之间的实时通信
